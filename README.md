# Tableau visualization exercises & dashboards

This repository contains a collection of Tableau workbooks developed for data visualization and analytics practice. The projects explore core Tableau concepts, including chart creation, continuous vs. discrete fields, date visualizations, Level of Detail (LOD) expressions, and interactive dashboards.

## Repository contents

| File | Description |
|------|-------------|
| `74513_PBMC_EJ01_812.twb` | Exercises demonstrating continuous and discrete dimensions, as well as date visualizations. |
| `74513_PBMC_EJ02_2P_812.twbx` | Additional Tableau exercises covering multiple visualization techniques and analytical scenarios. |
| `74513_PBMC_PR01_812.twbx` | Practical dashboard project featuring multiple charts and interactive visualizations. |
| `74513_MCPB_EJ013P_812.twbx` | Exercises focused on Level of Detail (LOD) expressions using **FIXED**, **INCLUDE**, and **EXCLUDE** calculations. |
| `74513_MCPB_PR02_812.twbx` | Dashboard project integrating multiple visualizations for business data analysis. |

## Topics covered

- Continuous vs. Discrete fields.
- Date visualizations.
- Bar, line, and other chart types.
- Interactive dashboards.
- Level of Detail (LOD) Expressions.
  - FIXED.
  - INCLUDE.
  - EXCLUDE.
- Business intelligence and data storytelling.

## Requirements

- Tableau Desktop (recommended version compatible with `.twb` and `.twbx` files).

## Purpose

These workbooks were created as hands-on exercises to strengthen Tableau skills, demonstrate best practices in data visualization, and apply business intelligence techniques through interactive dashboards.
